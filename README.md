# 🔬 DataLens – Smart Data Cleaning & Analysis Tool

DataLens is an intelligent web app built with Streamlit that allows users to upload datasets and automatically clean, analyze, and visualize them.

---

## 🚀 Features

* 📂 Upload CSV / Excel datasets
* 🧠 Automatic data cleaning (smart detection)
* 🔢 Convert messy columns to numeric
* ⚠️ Missing value detection & handling
* 📊 Data visualization (bar, scatter, heatmap, etc.)
* 📈 Column statistics & insights

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Streamlit
* Matplotlib, Seaborn

---

## ⚙️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run main.py
```
